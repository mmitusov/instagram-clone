Спева почистим проект и оставим только _app.tsx, index.tsx и глобальные стили.

Далее подумаем из каких основных частей будет состоять приложение. Будет состоять из хедера, фида, и модального окна. Начем с хедера.

Хедер будет состоять из 3-х частей: логоб поиск и меню иконок. При чем работая с формой поиска, если мы хотим задать ей стили при помощи tailwind, то нам нужно установить доп библиотеку - @tailwindcss/forms. Она позволяет перезаписывать стили форм/инпутов при помощи tailwind. Без нее этого сделать нельзя. После ее установки, ее нужно добавить в tailwind.config.js, поле - plugins: [require("@tailwindcss/forms")].
